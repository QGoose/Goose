# QuantumGoose

ICFP non-sense about quantum languages 

## Instruction

Compiling the project requires `opam` and `dune` to be installed

**To install opam**

**To install dune**
```
opam install dune
```
**To compile**

```
dune build

```

**To execute the QuantumGoose compiler (qgc)**

```
dune exec qgc
```
